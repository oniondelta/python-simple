## 臺灣銀行匯率轉換 Exchange Rate in Taiwan
#### 資料來源 臺灣銀行網站：https://rate.bot.com.tw/xrt?Lang=zh-TW
#### 匯率轉換有方向性，例如：現時匯率為 30 元整新台幣兌換 1 元整美金，但當你拿 1 元整美金，會換匯到少於 30 元整新台幣
#### 因此寫一個簡單 Python 程式，從台灣銀行爬蟲每日匯率，然後計算輸入金額（各方向匯兌的金額）
